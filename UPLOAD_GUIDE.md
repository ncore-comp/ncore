# GitHub 업로드 안내

이 폴더는 GitHub에 올릴 파일만 따로 모아둔 제출용 폴더입니다.

## 올리는 방법

1. GitHub 저장소 첫 화면으로 들어갑니다.
2. 루트에 있는 파일은 저장소 루트에 같은 이름으로 덮어씁니다.
3. `functions`, `js`, `scripts` 폴더 안 파일은 저장소의 같은 폴더에 덮어씁니다.
4. 폴더 이름은 바꾸지 말고 그대로 유지합니다.
