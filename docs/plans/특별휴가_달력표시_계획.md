# 특별휴가 달력 표시 계획
# Special Leave Calendar Display Plan

## 목표
## Goal

- 개인 달력에서 특별휴가 `부여 발생일`과 `사용 가능 기간`을 한눈에 보이게 한다.
- Make special leave `grant event dates` and `usable windows` visible at a glance on the personal calendar.

## 1차 범위
## Phase 1 Scope

- 개인 달력(`calendarMode = self`)만 적용
- Apply only to the personal calendar (`calendarMode = self`)

- 발생일은 셀 안 작은 칩으로 표시
- Show the event date as a small chip inside the day cell

- 사용 가능 기간은 셀 하단 얇은 막대바로 표시
- Show the usable window as a thin bar at the bottom of each day cell

- 막대바는 오늘 기준으로 경과 구간과 남은 구간을 시각적으로 다르게 표현
- Visually distinguish elapsed and remaining portions of the bar relative to today

## 표시 규칙
## Display Rules

- 발생일 당일은 사용 불가이므로 `발생` 칩만 표시하고 막대바는 표시하지 않는다.
- Because the event day itself is not usable, show only the `event` chip on that day and no bar.

- `usableFromDate ~ usableToDate` 구간에만 막대바를 표시한다.
- Show the bar only from `usableFromDate` to `usableToDate`.

- 한 날짜에 여러 특별휴가가 겹치면 여러 줄로 쌓는다.
- If multiple grants overlap on the same date, stack them in multiple rows.
