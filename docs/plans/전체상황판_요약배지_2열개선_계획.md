# 전체상황판 요약배지 2열 개선 계획

## 작업 일시
- 2026-04-01

## 작업 배경
- 전체상황판 월간 요약 달력에서 날짜칸 안 요약 배지가 세로 1열로 쌓이고 있습니다.
- 4월 1일처럼 항목이 많은 날은 글이 잘리거나 칸이 답답하게 보입니다.

## 목표
- 날짜칸 안 요약 배지를 `2열`로 배치해 잘림을 줄이고 가독성을 높입니다.

## 추천 방향
- 현재:
  - `flex-col`
  - 세로 1열 적층
- 변경:
  - `grid grid-cols-2`
  - 2열 진열 방식

## 적용 대상
- `전체상황판 > 월간 요약 달력`
- 날짜칸 안의 요약 배지
  - 연차
  - 반차
  - 시간차(퇴근)
  - 시간차(외출)
  - 장기부재
  - 대기

## 구현 기준
1. 요약 배지 컨테이너를 2열 grid로 변경
2. 배지 높이/패딩/폰트 크기를 약간 축소
3. 칸 안에서 더 많이 보이도록 여백 조정
4. 기존 색상 구분은 유지

## 기대 효과
- 4/1처럼 항목이 많은 날도 더 안정적으로 표시
- 잘림과 과한 세로 적층 완화
- 전체상황판 달력의 한눈에 보기 성능 개선
