# 권한설정 세부설정 분리 / 모바일 권한 추가 계획

## 목적

- 권한 탭의 `권한/설정 접근`, `구성원 관리`를 기존 달력/승인/팀원현황 그룹 바깥으로 분리한다.
- 새 그룹 이름은 `세부설정`으로 한다.
- 세부설정 안에서 PC용/모바일용 권한을 각각 분리한다.

## 구현 원칙

- UI만 바꾸는 것이 아니라 저장 구조와 실제 접근 판단까지 같이 분리한다.
- 기존 사용자 데이터는 호환성을 위해 예전 값으로 PC/모바일 권한을 동시에 해석한다.
- 마스터는 기존처럼 전체 허용 유지
