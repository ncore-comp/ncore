# 모바일 상단 드롭다운 / 달력 접기 계획

## 목적

- 모바일 화면에서 상단 관리자 버튼이 줄바꿈으로 흩어지는 문제를 줄인다.
- 모바일 화면에서 월간 달력이 화면을 크게 차지하는 문제를 줄인다.
- PC 화면은 유지하고 모바일 화면만 다듬는다.

## 변경 원칙

- PC 레이아웃은 건드리지 않는다.
- 모바일에서만 적용한다.
- 상단 버튼은 여러 개의 버튼 대신 드롭다운 1개로 정리한다.
- 달력은 팀원 연차 현황 카드처럼 접기/펼치기 구조로 바꾼다.

## 구현 항목

### 1. 모바일 상단 메뉴 드롭다운

- 대상: 상단 네비게이션의 관리자 버튼 묶음
- 방식:
  - `전체 상황판`
  - `권한/설정`
  - `구성원 관리`
  - `비번변경`
  - `로그아웃`
  를 모바일에서만 드롭다운 선택 항목으로 제공
- 결과:
  - 모바일 상단에는 이름/직급 정보와 드롭다운만 보이게 정리

### 2. 모바일 달력 접기/펼치기

- 대상: 대시보드의 월간 달력 카드
- 방식:
  - 모바일에서만 접기/펼치기 헤더 추가
  - 기본값은 접힘
  - 누르면 기존 월간/리스트 달력 내용을 그대로 보여줌
- 헤더 문구:
  - 현재 선택된 달력 탭 기준으로 `연차 현황 / 개인`, `연차 현황 / 매뉴얼팀` 식으로 표시

## 코드 예상 위치

## 검증 기준

1. PC 화면은 기존과 동일
2. 모바일 화면 상단 버튼이 드롭다운으로 정리됨
3. 모바일 달력이 접힌 상태로 시작함
4. 펼치면 기존 월간/리스트 기능이 정상 동작함
5. 로그인, 권한/설정, 구성원 관리, 로그아웃 이동이 정상 동작함
