# 전체상황판 시간차 분리 / 현재시각 자리배치 계획

## 목적
- `시간차(퇴근)`과 `시간차(외출)`을 별도 분류로 나눈다.
- 전체 자리배치도는 `오늘 날짜`일 때 현재 시각 기준으로 실제 자리에 있는지 없는지를 반영한다.

## 쉬운 비유
- 지금은 `시간차`라는 큰 상자에 `퇴근`과 `외출`이 같이 들어 있다.
- 이를 `시간차(퇴근)` 상자와 `시간차(외출)` 상자로 나누는 작업이다.
- 자리배치도는 지금 `오늘 일정표`만 보고 비움 처리하지만,
  앞으로는 `경비실 시계`까지 같이 보고 현재 시점 부재 여부를 판단하게 바꾸는 작업이다.

## 작업 1. 전체상황판 분류 세분화

### 현재 상태
- `반차`
- `시간차`
로만 나뉘어 표시된다.

### 변경 방향
- 아래 4개 축으로 분리
  - `연차`
  - `반차`
  - `시간차(퇴근)`
  - `시간차(외출)`

### 적용 위치
1. 전체상황판 요약 패널
2. 월간 요약 달력 칩
3. 우측/상세 패널 일반 부재 목록

## 작업 2. 자리배치도에 현재 시각 개념 반영

### 현재 상태
- 같은 날짜면 시간 구간과 상관없이 부재 처리되는 항목이 있다.

### 변경 방향
- `오늘 날짜`를 볼 때만 현재 시각 기준으로 실제 부재 여부를 판단

### 판정 원칙
1. `연차`, `특별휴가`, `장기부재`
   - 하루 종일 부재
2. `반차(오전)`
   - 오전 시간대만 부재
3. `반차(오후)`
   - 오후 시간대만 부재
4. `시간차(퇴근)`
   - `timeRange` 시작 시각 이후부터 부재
5. `시간차(외출)`
   - `timeRange` 구간 동안만 부재

### 날짜별 원칙
- `오늘`
  - 현재 시각 기준 판정
- `과거`
  - 그 날짜의 일정 기록 기준으로 그대로 표시
- `미래`
  - 예정된 부재 기준으로 그대로 표시

## 작업 3. 색상 분리

### 추천
- `반차`
  - 주황 계열
- `시간차(퇴근)`
  - 청록/시안 계열
- `시간차(외출)`
  - 보라/자수정 계열 또는 다른 차가운색
