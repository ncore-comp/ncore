# 모바일 달력 상세보기 / 최근 신청 내역 접기 계획

## 목적

- 모바일 달력 칸 안의 작은 일정 텍스트를 억지로 읽지 않도록 바꾼다.
- 날짜를 터치하면 해당 날짜의 상세내역을 한 번에 읽을 수 있게 한다.
- 최근 신청 내역도 모바일에서는 접기/펼치기 구조로 바꿔 화면 길이를 줄인다.
- PC 화면은 유지하고 모바일만 개선한다.

## 변경 방향

### 1. 모바일 달력 날짜 터치 상세보기

- 모바일 달력 칸에는 요약만 보이게 유지
- 날짜를 누르면 하단 시트 또는 모달 형태로 상세내역 표시
- 상세내역 항목:
  - 날짜
  - 신청자명
  - 종류
  - 시간 범위
  - 사유
  - 상세사유/반려사유가 있으면 함께 표시

### 2. 모바일 최근 신청 내역 접기

- 최근 신청 내역은 모바일에서만 접기/펼치기 카드로 변경
- 기본값은 접힘
- 펼치면 기존 목록/페이지네이션을 그대로 사용

## 구현 원칙

- 모바일에서만 적용
- PC `renderCal`, `renderRecentRequestList` 구조는 유지
- 기존 달력 더블클릭 신청 로직은 건드리지 않음
- 모바일 날짜 터치는 `상세보기` 우선

2. 모바일에서 날짜 칸 안 가독성이 이전보다 좋아진다
3. 최근 신청 내역은 기본 접힘으로 시작한다
4. 펼치면 기존 목록/페이지네이션이 정상 동작한다
5. PC 화면은 기존과 동일하다
