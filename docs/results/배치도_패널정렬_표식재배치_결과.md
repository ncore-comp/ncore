# 배치도 패널 정렬 및 표식 재배치 결과

- 작업일시: 2026-04-01 11:52:31
- 대상 화면: 전체 상황판 > 날짜 클릭 > 자리 배치도 모달

## 요청 사항

1. `당일 부재 인원` 패널 테두리를 왼쪽 자리 배치도 테두리 하단과 맞춤
2. `대표님실` 라벨을 우측 사각영역의 가로/세로 중앙으로 이동
3. `입구` 화살표 제거
4. `입구` 텍스트를 좌측 위/아래 사각영역 사이 흰색 여백 중앙으로 이동

## 적용 내용

- 오른쪽 패널에 `seat-map-side-panel` 클래스를 부여
- 자리 배치도(`seat-map-stage`) 실제 높이를 읽어 오른쪽 패널 높이를 동일하게 맞추는 `syncSituationBoardSeatMapPanelHeight()` 추가
- 모달 오픈 직후 `requestAnimationFrame`과 짧은 `setTimeout`으로 높이 동기화 수행
- `입구` 표식은 화살표 제거 후, 흰색 여백 중앙에 라벨만 중앙 정렬
- `대표님실` 라벨은 우측 하단 사각영역의 중앙으로 이동

## 기대 효과

- 오른쪽 `당일 부재 인원` 패널의 하단 테두리가 왼쪽 자리 배치도와 일치함
- 부재 인원 스크롤은 유지하면서 전체 레이아웃이 안정적으로 보임
- `입구`, `대표님실` 위치 의미가 더 직관적으로 전달됨
