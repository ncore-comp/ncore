# 모바일 상단 드롭다운 / 달력 접기 결과

## 작업 결과

- 모바일 상단 관리자 버튼 묶음을 드롭다운 메뉴로 변경했다.
- 별도 `관리 메뉴` 셀렉트 박스를 제거하고, 이름/직급 영역 클릭형 메뉴로 다듬었다.
- 프로필 아래 `관리 메뉴` 문구는 제거하고 우측 화살표 아이콘만 남겼다.
- 모바일 월간 달력 카드를 접기/펼치기 구조로 변경했다.
- 모바일 `내 연차 현황` 제목은 줄바꿈이 나지 않도록 글자 크기를 줄이고 한 줄 고정 처리했다.
- PC 화면은 유지했다.
