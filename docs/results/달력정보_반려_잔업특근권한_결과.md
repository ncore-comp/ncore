# 달력정보 반려/잔업특근 권한 추가 결과

## 작업 일시
- 2026-03-31

## 반영 내용
- `권한/설정 > 권한 탭 > 달력정보`에 아래 체크박스 2개를 추가했습니다.
  - `반려`
  - `잔업/특근`
- 체크된 사람만 달력 상단 탭에서 해당 항목이 보이도록 연결했습니다.
- `반려` 탭은 기존 승인권한 유무와 분리하고, `달력정보 > 반려` 권한으로만 노출되게 변경했습니다.
- `잔업/특근` 탭을 추가하고, 체크된 사람만 잔업/특근 보고 기록을 달력에서 볼 수 있게 변경했습니다.

## 동작 기준
- `반려` 탭:
  - 사용자가 달력으로 볼 수 있는 범위(`개인/매뉴얼팀/파츠북팀/모두`) 안에서 `rejected` 상태 요청만 표시
- `잔업/특근` 탭:
  - 사용자가 달력으로 볼 수 있는 범위 안에서 `reported` 상태의 `잔업`, `특근` 요청만 표시
- 마스터:
  - 새 권한 2개도 자동으로 모두 체크 상태 유지

## 저장 구조
- 클라이언트/서버 권한 객체에 아래 필드 추가
  - `calendarRejected`
  - `calendarWorkReport`
- 사용자 저장 시 함께 저장되도록 반영
