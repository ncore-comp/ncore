# 전체상황판 당일 부재 인원 패널 확장 결과

- 작업일시: 2026-04-01 11:05:35
- 대상 화면: 전체 상황판 > 날짜 클릭 > 자리 배치도 모달

## 문제

- 오른쪽 `당일 부재 인원` 패널 폭이 너무 좁고, 본문 전체 크기에 비해 의미 없이 작게 보였다.
- 부재 기록이 여러 줄로 늘어나도 패널은 작은 사이드바처럼 고정돼 가독성이 떨어졌다.

## 원인

- 자리 배치도 모달 레이아웃이 `xl:grid-cols-[minmax(0,1fr)_320px]`로 고정되어 있었다.
- 오른쪽 목록도 `max-h-[620px]`로 제한돼, 모달 전체 높이를 충분히 활용하지 못했다.

## 개선 내용

- 오른쪽 패널 고정 폭을 `320px`에서 `380px`로 확장
- 레이아웃에 `items-stretch`를 적용해 본문 높이에 맞춰 패널도 함께 늘어나도록 조정
- 목록 카드 컨테이너를 `flex` 구조로 바꿔, 내부 스크롤 영역이 패널 전체 높이를 자연스럽게 사용하도록 개선

## 기대 효과

- `당일 부재 인원` 목록이 더 넓고 안정적으로 보인다.
- 직원 카드와 복수 부재 기록이 덜 답답하게 표시된다.
- 전체 자리 배치도와 균형이 더 자연스러워진다.
