# 잔업특근 상세모달 확장 결과

## 작업 시각
- 2026-04-01 13:15:21

## 문제
- `월별 잔업/특근 현황 > 상세보기` 모달이 좁고 낮아서
  - `비고` 열 내용이 가로로 잘리고
  - 행 수가 많을 때 아래쪽 내용이 세로로도 잘렸습니다.

쉽게 말하면:
- 긴 장부를 작은 서랍에 넣어둔 상태였습니다.

## 조치
- `보고 상세` 모달 폭 확대
  - `max-w-2xl` -> `max-w-[1280px]`
- 모달 최대 높이 제한
  - `max-h-[88vh]`
- 모달 본문에 내부 스크롤 추가
  - `report-detail-body` -> `overflow-auto`
- 관리자용 상세 테이블 최소폭 확대
  - `min-w-[760px]` -> `min-w-[1100px]`
- `비고` 셀 줄바꿈 강화
  - `break-all` 추가

## 결과
- 상세 모달이 더 넓게 열립니다.
- 데이터가 많아도 모달 내부에서 위아래 스크롤됩니다.
- 긴 `비고` 문자열도 이전보다 덜 잘리고 더 많이 보입니다.
