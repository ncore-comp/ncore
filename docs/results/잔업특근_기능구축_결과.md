## 동작 규칙
### 잔업
- 직원의 분기별 퇴근시간을 시작 시간으로 자동 적용
- 1시간 단위 보고 시간 선택
- `reported` 상태로 즉시 저장

### 특근
- 시작 시간 직접 선택
- 1시간 단위 보고 시간 선택
- `reported` 상태로 즉시 저장

## 제외 처리
- 연차 차감 게이지 제외
- 승인/반려/취소 흐름 제외
- 달력 부재 표시 제외
- 전체 상황판 부재 계산 제외
