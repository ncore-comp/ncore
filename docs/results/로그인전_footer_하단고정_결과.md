# 로그인 전 footer 하단 고정 결과

- 작업일시: 2026-04-01 12:20:16
- 대상 화면: 로그인 전 화면만

## 문제

- 로그인 전 첫 화면에서 하단 문구(`Designed & Developed by NCORE`, `Copyright`)가
  첫 화면에 보이지 않고 스크롤을 내려야만 보였다.

## 원인

- 공통 `main`이 `pt-24`, `pb-12`를 가지고 있고,
- 로그인 카드 래퍼가 `min-h-[calc(100vh-160px)]`로 거의 한 화면 높이를 다시 차지하고 있었다.
- 그 결과 footer가 일반 문서 흐름 아래로 밀려나 로그인 전 첫 화면 밖으로 내려갔다.

## 개선 내용

- 로그인 전 화면 전용 `login-screen-active` body 클래스를 추가
- 로그인 전일 때만 footer를 브라우저 하단 고정형으로 표시
- 로그인 카드 래퍼 최소 높이는 `100vh-250px` 수준으로 줄여 footer가 첫 화면 안에 자연스럽게 들어오도록 조정
- 로그인 후 화면에서는 `login-screen-active`를 해제하여 기존 레이아웃 유지

## 중요 사항

- 로그인 후 화면은 변경하지 않음
- 로그인 전 로비 화면에서만 적용되는 레이아웃 분기
