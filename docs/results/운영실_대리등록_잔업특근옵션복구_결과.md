# Admin Proxy Request Work Report Options Restore Result

- 작성 일자: 2026-04-06
- 버전: `v119`

## 수정 내용
## What Changed

- `관리자 대리 등록` 모달(`req-modal`)의 종류 드롭다운에 `잔업`, `특근` 옵션을 다시 연결했습니다.
- Restored `잔업` and `특근` options in the admin proxy request modal (`req-modal`) type dropdown.

- 기존 저장 로직은 이미 잔업/특근 분기를 가지고 있었기 때문에, 이번 수정은 `종류 목록 연결` 복구에 집중했습니다.
- The save logic already supported work report branches, so this change focused on reconnecting the type list.

## 원인
## Root Cause

- 관리자 대리등록은 `req-modal`을 사용하지만, `getRequestTypeOptionsHtml()`이 연차/반차/시간차/특별연차만 반환하고 있었습니다.
- Admin proxy registration uses `req-modal`, but `getRequestTypeOptionsHtml()` only returned leave/time-off types and special leave.

- 그 결과 `잔업/특근` 저장 엔진은 존재해도, 관리자 화면에서는 해당 종류를 고를 수 없었습니다.
- As a result, the work report save engine existed, but admins could not select those types from the UI.
