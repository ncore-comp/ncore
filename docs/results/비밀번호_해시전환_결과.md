# 비밀번호 해시 전환 결과

## 1. 목적

- 평문 비밀번호 저장을 중단하고 해시 기반 인증으로 전환
- 테스트 기간 정책에 따라 전 직원 초기 비밀번호를 `0`으로 통일
- DB에는 평문이 아닌 해시만 남기도록 변경

쉽게 말하면:
- 모든 계정 자물쇠를 새 규격으로 교체했고,
- 테스트 기간이라 임시키는 모두 `0`으로 맞춘 상태다.
