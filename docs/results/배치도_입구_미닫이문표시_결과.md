# 배치도 입구/미닫이문 표시 결과

- 작업일시: 2026-04-01 11:42:21
- 대상 화면: 전체 상황판 > 날짜 클릭 > 자리 배치도 모달

## 요청 사항

- A 위치에 `입구` 표시
- B 위치에 `미닫이문` 표기

## 적용 방식

- 원본 평면도 이미지는 수정하지 않음
- 자리 배치도 오버레이 위에 안내 표식을 덧붙이는 방식으로 구현

## 반영 내용

- `입구`
  - 좌측 통로 영역에 `입구` 라벨과 방향 화살표 배치
- `미닫이문`
  - 우측 통로 영역에 미닫이문 기호(문틀 선 + 문짝 표시)와 `미닫이문` 라벨 배치

## 기대 효과

- 사용자가 자리 배치도를 볼 때 출입 위치를 직관적으로 이해할 수 있음
- 평면도 원본을 건드리지 않아 위치/모양 수정이 쉬움
