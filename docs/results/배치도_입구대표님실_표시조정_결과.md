# 배치도 입구/대표님실 표시 조정 결과

- 작업일시: 2026-04-01 11:45:54
- 대상 화면: 전체 상황판 > 날짜 클릭 > 자리 배치도 모달

## 조정 내용

- `입구` 표식을 기존보다 아래(A 위치)로 이동
- 우측 안내 표식은 `미닫이문` 기호를 제거
- 대신 `대표님실` 텍스트 라벨만 남기도록 변경

## 적용 방식

- 원본 평면도 이미지는 그대로 유지
- 오버레이 좌표만 조정해서 안내 표식을 재배치

## 기대 효과

- `입구`가 실제 진입 동선처럼 더 자연스럽게 보임
- 우측 공간 의미가 `대표님실`로 더 명확하게 전달됨
- 과한 기호 없이 단순하고 읽기 쉬운 배치도로 정리됨
