# 잔업특근 역할별현황 결과

## 작업 일시
- 2026-04-01

## 반영 내용
- `잔업/특근 현황`을 역할 기준으로 두 가지 버전으로 분리했습니다.

### 1. 개인용 현황
- 일반 직원은 본인 기록만 봅니다.
- 상단 total:
  - `내 총 잔업시간`
  - `내 총 특근일수`
- 하단 테이블:
  - `일자`
  - `시간`
  - `구분`
  - `사유`
  - `상세`

### 2. 관리자용 현황
- 관리자/운영자/대표/마스터는 직원별 집계표를 먼저 봅니다.
- 상단 total:
  - `총 잔업시간`
  - `총 특근일수`
- 하단 집계 테이블:
  - `직원명`
  - `총 잔업시간`
  - `총 특근일수`
  - `상세`
- `상세`를 누르면 해당 직원의 정산월 날짜별 상세내역이 별도 모달로 열립니다.

## 관리자용 상세 모달
- `직원명 + 정산월` 기준으로 표시
- 상단에 해당 직원 소계 표시
  - `총 잔업시간`
  - `총 특근일수`
- 하단에 날짜별 상세 원장 표시
  - `일자`
  - `시간`
  - `구분`
  - `사유`
  - `요청 부서`
  - `비고`

## 정산 기준
- 계속 `16일 ~ 다음달 15일` 기준 유지
- 예:
  - `2026년 3월 정산 = 2026-02-16 ~ 2026-03-15`

## 관리자 판정 기준
- 아래 중 하나면 관리자용 현황을 봅니다.
  - `master`
  - `ceo`

  - `운영실 접근 권한`
