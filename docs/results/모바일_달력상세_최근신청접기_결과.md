# 모바일 달력 상세보기 / 최근 신청 내역 접기 결과

## 작업 결과

- 모바일 달력 날짜를 터치하면 상세내역 시트가 열리도록 변경했다.
- 모바일 달력 칸 안에는 사용자 이름 대신 해당 날짜의 총 건수만 보이게 단순화했다.
- 모바일 최근 신청 내역을 접기/펼치기 카드로 변경했다.
- 날짜 상세 시트에서 누락되어 있던 상태 텍스트/상태 색상 도구를 보강했다.
- 모바일 `내 연차 현황`의 남은 시간은 한 줄 고정으로 다시 보정했다.
- 모바일 상단 프로필 메뉴는 상단바 높이를 늘리지 않는 팝오버 구조로 다시 보정했다.
- 모바일 날짜 상세 시트가 열릴 때 배경 화면 스크롤을 잠그도록 보정했다.
- 모바일 날짜 상세 시트 내부에 `overscroll-contain`을 적용해 배경으로 스크롤이 새지 않게 했다.
- 모바일 날짜 상세 시트는 `내용만큼 뜨는 카드`에서 `고정 높이 bottom sheet` 구조로 변경했다.
- PC 화면은 유지했다.

## 주요 변경

### 1. 모바일 달력 날짜 상세보기

- 날짜 터치 시 하단 시트 형태의 상세내역 박스 표시
- 상세내역 항목:
  - 날짜
  - 신청자명
  - 종류
  - 시간 범위
  - 사유
  - 상세사유/반려사유

### 2. 모바일 달력 칸 요약 단순화

- 기존: 칸 안에 사용자명/종류를 작게 직접 표시
- 변경: 모바일에서는 `N건` 요약만 표시

### 3. 모바일 최근 신청 내역 접기

- 기본값: 접힘
- 필요 시 펼쳐서 기존 목록/페이지네이션 사용
