# 홈 대시보드 팀원 연차 현황 독립 접기 v152 결과
# Home Dashboard Member Status Independent Collapse v152 Result

## 요약
## Summary

- 홈화면의 `팀원 연차 현황`을 부서별 독립 접기 구조로 변경했다.
- The home dashboard `Team Member Leave Status` now uses independent collapse sections per department.

- `파츠북팀`과 `매뉴얼팀`은 각각 따로 펼치고 접을 수 있다.
- `Partsbook Team` and `Manual Team` can now be expanded and collapsed independently.

- 기본 상태는 두 부서 모두 접힘이다.
- Both department sections are collapsed by default.

## 반영 내용
## Applied Changes

- 대시보드 스크립트를 `ui_dashboard_v152_codex.js`로 올렸다.
- The dashboard script was bumped to `ui_dashboard_v152_codex.js`.

- 모바일 전용 접기였던 기존 조건을 제거하고, 화면 크기와 관계없이 부서별 토글을 사용하도록 바꿨다.
- The old mobile-only collapse condition was removed so department toggles now work regardless of screen size.

- 접기 상태는 부서명 기준으로 독립 저장된다.
- Collapse state is stored independently per department name.

## 확인 포인트
## Checkpoints

1. 홈화면에서 `파츠북팀`과 `매뉴얼팀` 카드가 처음에는 접혀 있어야 한다.
1. On the home dashboard, both `Partsbook Team` and `Manual Team` cards should start collapsed.

2. `파츠북팀`을 펼쳐도 `매뉴얼팀`은 그대로 접혀 있어야 한다.
2. Expanding `Partsbook Team` should not auto-expand `Manual Team`.

3. `매뉴얼팀`을 펼쳐도 `파츠북팀` 상태는 유지되어야 한다.
3. Expanding `Manual Team` should not change the `Partsbook Team` state.
