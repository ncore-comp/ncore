# 전체상황판 자리배치도 당일기준 v151 결과
# Situation Board Seat Map Same-Day Basis v151 Result

- 문제
- Issue
  - 전체 상황판 자리배치도는 오늘 날짜일 때 현재 시각 기준으로 반차/시간차(퇴근/외출) 부재 여부를 계산했습니다.
  - On today's date, the situation-board seat map calculated half-day and time-off absence using the current clock time.

  - 그래서 같은 날 승인된 `시간차(퇴근)` 요청이 있어도 아직 그 시각이 되지 않으면 자리배치도와 우측 부재 인원 목록에 보이지 않았습니다.
  - As a result, an approved `time-off (leave early)` request could be invisible in the seat map and right-side absent list until the current time reached the request window.

- 변경
- Change
  - `ui_shared_helpers_v151_codex.js`의 `getSituationBoardRequestPresenceState()`에서 오늘 날짜 특수 분기를 단순화했습니다.
  - Simplified the today-specific branch inside `getSituationBoardRequestPresenceState()` in `ui_shared_helpers_v151_codex.js`.

  - 오늘 날짜라도 `approved / cancel_requested` 상태인 요청은 현재 시각과 무관하게 `당일 부재`로 본다.
  - Even on the current day, requests in `approved / cancel_requested` state are now treated as `same-day absent` regardless of the current time.

  - `pending` 요청은 계속 실제 부재로 보지 않는다.
  - `pending` requests are still not treated as actual absences.

- 기대 결과
- Expected result
  - 자리배치도는 `현시간 기준`이 아니라 `선택한 날짜 기준`으로 부재를 보여야 합니다.
  - The seat map should now reflect absence based on the `selected date`, not the current time.

  - 오늘 날짜의 승인된 반차/시간차(퇴근/외출)도 자리배치도와 우측 `당일 부재 인원` 목록에 표시되어야 합니다.
  - Approved half-day and time-off requests on today should appear in both the seat map and the right-side `same-day absent` list.
