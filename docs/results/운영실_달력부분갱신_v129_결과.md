# Admin Ops Calendar Partial Refresh v129 Result

## 목적
## Purpose

- 운영실 달력 날짜 클릭 시 권한/설정 페이지 전체가 다시 그려져 생기던 깜빡임을 줄입니다.
- Reduce the flicker caused by rerendering the entire permissions page when clicking a date in the admin ops calendar.

## 핵심 변경
## Key Change

- 날짜 선택, 대상 직원 변경, 월 이동, 로그 새로고침은 `권한/설정 전체 렌더` 대신 `운영실 패널 루트만 갱신`하도록 변경했습니다.
- Changed date selection, target-user change, month navigation, and log refresh to update only the admin-ops panel root instead of rerendering the entire permissions page.

## 기대 효과
## Expected Effect

- 운영실 달력 날짜 클릭 시 상단 네비와 권한/설정 전체 박스가 다시 그려지지 않습니다.
