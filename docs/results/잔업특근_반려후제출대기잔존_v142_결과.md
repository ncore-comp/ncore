# 잔업/특근 반려 후 제출 대기 잔존 v142 결과
# Overtime/Holiday Work Rejection Still Showing as Pending v142 Result

## 요약
## Summary

- `잔업/특근 제출 대기`에 반려된 건이 계속 남는 문제를 수정했다.
- Fixed the issue where rejected work report settlements still remained in `Pending Overtime/Holiday Work Submission`.

- 핵심 원인은 서버 응답에서 `settlementStatus` 등 정산 상태 필드가 누락되어, 프런트가 `settlementSubmittedAt`만 보고 `submitted`로 오인하던 구조였다.
- The root cause was that settlement status fields such as `settlementStatus` were missing from the server response, so the frontend incorrectly treated any item with `settlementSubmittedAt` as `submitted`.

## Applied Changes

## Expected Result
