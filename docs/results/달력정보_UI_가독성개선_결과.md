# 달력정보 UI 가독성 개선 결과

## 작업 일시
- 2026-04-01

## 반영 내용
- `권한/설정 > 권한 탭 > 달력정보` 체크박스 6개를 `2줄 3칸` 구조로 유지하면서,
  상단 헤더 라벨 박스는 제거했습니다.
- 직원 행은 줄무늬처럼 아주 옅은 명암을 넣어, 행 구분이 더 잘 되도록 보완했습니다.

## 변경 사항
### 1. 상단 헤더 박스 제거
- 기존에는 `자기연차 / 모두 / 매뉴얼팀 / 반려 / 파츠북팀 / 잔업/특근` 라벨 박스가
  체크박스 위에 한 번 더 표시되고 있었습니다.
- 현재는 이 상단 라벨 박스를 제거해, 화면이 덜 복잡하게 보이도록 정리했습니다.

### 2. 직원 행 명암 보강
- 일반 직원 행은 `흰색 / 옅은 명암`이 번갈아 보이도록 조정했습니다.
- 대표 / 팀리더 / 파트리더 강조 행은 기존 강조 색상을 그대로 유지했습니다.

## 현재 구조
- 1열:
  - `자기연차`
  - `모두`
- 2열:
  - `매뉴얼팀`
  - `반려`
- 3열:
  - `파츠북팀`
  - `잔업/특근`

## 개선 효과
- 달력정보 영역이 더 단순하게 보임
- 가로 스크롤 없이 한눈에 보기 쉬움
- 직원 행이 살짝 구분되어 시선 추적이 편해짐
