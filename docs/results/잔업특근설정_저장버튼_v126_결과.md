# Workreport Settings Save Button v126 Result

## 요약
## Summary
- `잔업/특근 설정` 탭 하단에 `설정 저장` 버튼이 다시 보이도록 복구했습니다.
- The `설정 저장` button is now visible again at the bottom of the `잔업/특근 설정` tab.

- 저장 로직은 원래 있던 `app.saveMasterSettings()`를 그대로 사용하고, 버튼 노출 조건만 보강했습니다.
- The existing `app.saveMasterSettings()` logic was preserved; only the button visibility condition was fixed.

## 원인
## Cause
- `saveMasterSettings()`는 `settings`와 `workreport` 둘 다 처리하고 있었습니다.
- `saveMasterSettings()` already handled both `settings` and `workreport`.

- 하지만 하단 버튼 렌더는 `settings` 탭만 `설정 저장`을 보이게 하고 `workreport` 탭은 빠져 있었습니다.
- However, the bottom button render condition only showed `설정 저장` for the `settings` tab and omitted the `workreport` tab.

## 수정 내용
## Applied Change
- `activeTab === 'settings' || activeTab === 'workreport'`
- The save-button condition now includes both tabs:
  - `settings`
  - `workreport`
