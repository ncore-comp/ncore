# 특별휴가 달력 표시 v145 결과
# Special Leave Calendar Display v145 Result

## 요약
## Summary

- 개인 달력에만 특별휴가 `발생일` 칩과 `사용 가능 기간` 막대바를 추가했다.
- Added a special-leave `event-date` chip and `usable-window` bar to the personal calendar only.

- 발생일 당일은 `발생` 칩만 표시하고, 사용 가능 기간 바는 표시하지 않는다.
- On the event day itself, only the `event` chip is shown, and no usable-window bar is drawn.

- 사용 가능 기간(`usableFromDate ~ usableToDate`)에는 날짜 셀 하단 얇은 막대바를 표시한다.
- During the usable window (`usableFromDate ~ usableToDate`), a thin bar is rendered at the bottom of each day cell.

- 막대바는 오늘 기준으로 지난 구간과 남은 구간의 농도를 다르게 표현한다.
- The bar uses different opacity for elapsed and remaining portions relative to today.

## 1차 범위
## Phase 1 Scope

- `calendarMode = self` 에서만 표시
- Shown only when `calendarMode = self`

- 팀/전체/반려/잔업특근 달력에는 아직 표시하지 않음
- Not yet shown on team/all/rejected/workreport calendars
