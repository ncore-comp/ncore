# 운영실 선택 날짜 대리 등록 버튼 복구 v127 결과
# Admin Ops Selected-Date Proxy Registration Button Restore v127 Result

## 요약
## Summary
- `운영실 > 선택 날짜로 대리 등록` 버튼이 눌러도 반응이 없던 원인은, 클릭 핸들러 함수 `openAdminProxyRequestModal()` 이 빠져 있었기 때문입니다.
- The reason the `운영실 > 선택 날짜로 대리 등록` button did nothing was that its click handler `openAdminProxyRequestModal()` was missing.

- 예전 정상 버전의 동작과 같은 역할을 하는 함수를 `v127`에 다시 복구했습니다.
- The function was restored in `v127` using the same behavior as the last known good version.

## 원인
## Cause
- 버튼 HTML은 존재했지만, `onclick="app.openAdminProxyRequestModal(...)"` 를 처리할 함수가 없었습니다.
- The button HTML existed, but there was no function to handle `onclick="app.openAdminProxyRequestModal(...)"`.

- 그래서 브라우저에서는 누를 수는 있지만 실제로 등록 모달을 열지 못했습니다.
- So the browser allowed the click, but could not open the proxy registration modal.
