# 특별휴가 종류 2단 레이아웃 v142 결과
# Special Leave Type Two-Row Layout v142 Result

## 요약
## Summary

- 특별휴가 종류 설정 카드를 한 줄 고정형 grid에서 2단 폼 레이아웃으로 변경했다.
- The special leave type settings card was changed from a single-row fixed grid to a two-row form layout.

- 윗줄에는 `특별휴가 이름 / 기본 일수 / 사용 타입 / 차감 방식`을 배치했다.
- The top row now contains `name / grant days / request mode / day count mode`.

- 아랫줄에는 `발생일 기준 / 소멸 방향 / 기한 일수 / 주말·공휴일 허용 / 사용 / 삭제`를 배치했다.
- The bottom row now contains `event-date basis / expiry direction / expiry days / holiday request / enabled / delete`.

## 기대 효과
## Expected Effects

- 긴 라벨이 앞칸 폭을 잡아먹는 현상을 줄인다.
- This reduces the way long labels were collapsing the first field.

- `결혼(본인)` 같은 이름이 잘리거나 `결:`처럼 눌려 보이던 문제를 줄인다.
- This reduces the clipping that made names like `결혼(본인)` appear truncated.

- 토글과 삭제 버튼의 세로 찌그러짐을 줄인다.
- This reduces the vertical squeezing of the toggle and delete button.
