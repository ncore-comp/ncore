# Special Leave Consumed Entry Display on Member Card v144 Result

## 요약
## Summary

- 특별휴가 `grantStatus` 판정 순서를 조정했다.
- Adjusted the priority order of special leave `grantStatus`.

- 이제 `availableHours <= 0` 이면 `eventDate` 누락보다 먼저 `consumed`로 판정한다.
- Now, if `availableHours <= 0`, the grant is treated as `consumed` before checking missing `eventDate`.

- 따라서 발생일이 비어 있어도 이미 다 사용한 특별휴가는 직원카드 이력에 표시된다.
- Therefore, even if `eventDate` is empty, a fully used special leave grant is shown in the member-card history.

- 동시에 `팀원 연차 현황` 집계에서는 `consumed / expired / inactive`를 모두 제외한다.
- At the same time, the team annual summary now excludes `consumed / expired / inactive`.
