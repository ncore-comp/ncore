## English
- Issue: #2
- Root cause:
  - `main#app-container` already had top padding
  - `.situation-board-page` also adds vertical padding
  - but `.situation-board-shell` did not subtract enough of that space, so the document became taller than the viewport
- Actual fix:
  - [index.html](/D:/MYDOC/Desktop/vscode/Web/firebase/ncore_web/ncore-main/index.html)
  - `.situation-board-shell`
    - `height: calc(100vh - 6.2rem)` -> `calc(100vh - 7.6rem)`
    - `min-height: calc(100vh - 6.2rem)` -> `calc(100vh - 7.6rem)`
- Result:
  - removed the meaningless main vertical scrollbar
- Numeric verification:
  - `innerHeight: 1200`
  - `docScrollHeight: 1200`
  - `hasVerticalScroll: false`
