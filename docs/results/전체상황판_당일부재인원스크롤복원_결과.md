# 전체상황판 당일 부재 인원 스크롤 복원 결과

- 작업일시: 2026-04-01 11:18:59
- 대상 화면: 전체 상황판 > 날짜 클릭 > 자리 배치도 모달

## 문제

- 오른쪽 `당일 부재 인원` 패널을 키운 뒤 내부 스크롤이 사라졌다.
- 부재 인원이 많아질수록 오른쪽 패널이 내용 길이만큼 커졌고,
  그 영향으로 왼쪽 자리 배치도 레이아웃도 함께 틀어졌다.

## 원인

- 오른쪽 패널을 `h-full`, `items-stretch` 기반으로 늘리면서,
  패널 내용 높이가 전체 grid row 높이를 끌어올리는 구조가 되었다.
- 이로 인해 왼쪽 `seat-map-stage`도 같은 줄 높이에 맞춰 늘어나 배치가 깨졌다.

## 개선 내용

- grid 정렬을 `items-start`로 되돌려 왼쪽 자리 배치도 영역이 내용 길이에 영향을 받지 않도록 수정
- 오른쪽 패널은 `xl:h-[78vh]`, `xl:max-h-[920px]` 고정 높이 영역 안에서만 커지게 변경
- 내부 목록은 `flex-1 + overflow-y-auto`로 다시 스크롤되도록 복원

## 기대 효과

- 당일 부재 인원이 많아도 오른쪽 패널 안에서만 스크롤된다.
- 왼쪽 자리 배치도 크기와 비율이 유지된다.
- 전체 모달 레이아웃이 부재 인원 수에 따라 틀어지지 않는다.
