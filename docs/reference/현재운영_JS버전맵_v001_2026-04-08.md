## 이후 버전 규칙
## Future Version Rule

- 다음 실제 운영 수정부터는 `v002`를 사용한다.
- Start with `v002` for the next real production change.

- 같은 릴리즈에서 수정한 운영 파일은 가능한 한 같은 버전 번호로 맞춘다.
- Files changed in the same release should use the same version number whenever possible.

- 이전 운영 세트는 `js\legacy_versions`로 이동한다.
- Previous production sets move into `js\legacy_versions`.
