# 빠른 시작 체크리스트

## 1. 일반 직원

- 로그인 가능 여부 확인
- 내 연차 현황 확인
- 연차 신청 1회 해보기
- 최근 신청 내역 확인
- 모바일에서 날짜 상세내역 열어보기

## 2. 승인권자

- 결재 대기 목록 확인
- 승인 1건 처리
- 반려 1건 처리
- 팀원 연차현황 확인
- 전체 상황판 확인

## 3. 대표

- 전체 상황판 접근 권한 확인
- 자리배치도 확인
- 날짜별 부재 요약 확인
- 모바일에서도 동일 권한 확인

## 4. 운영자

- 권한/설정 접근 확인
- 구성원 관리 접근 확인
- 운영실 접근 확인
- 수동 휴일 등록/삭제 확인
- 공휴일 동기화 범위 확인
- 로그 CSV 다운로드 확인

## 5. 시연 전 체크

- 시연 계정 로그인 확인
- 테스트 데이터 상태 확인
- 공휴일 동기화 정상 여부 확인
- 운영실 주요 버튼 동작 확인
- 모바일 화면에서 상단 메뉴/달력 상세 확인

## 6. 운영 전 체크

- 권한표 저장 상태 확인
- 대표/팀장/서무 권한 분리 확인
- 분기별 근무시간 입력 확인
- 백업 최신본 존재 확인

## 7. 한 줄 정리

- 직원은 `신청`
- 승인권자는 `결정`
- 대표는 `전체 파악`
- 운영자는 `보정과 관리`
